# Avo Toolkit for Zotero — distribution

This public repo hosts the over-the-air update files for the Avo Toolkit Zotero plugin:

- `update.json` (on `main`) — the manifest Zotero polls.
- GitHub Releases — the installable `avo-toolkit.xpi` (minified).

The plugin **source is private**. End users install `avo-toolkit.xpi` once and receive
updates automatically.
